About the Teachscape Coding Challenge

Build and test a simple web-based system that takes the given CSV data (see below) containing contact information in structured format and translates it into Contacts and Companies. The system should store the data in a database and allow for access with a brower. The system should have at least three screens

  1. File upload
    2. Screen for selection and uploading the CSV from a file
  2. List view for Contacts
    3. Screen showing the data that was extracted by the upload resides inside the system. The screen should show the uploaded data in a table-like format with (optionally) sortable columns.
  3. List view for Companies
    4. Screen like 2) but showing the extracted companies and their details
  4. (Optional )
    5. Use one of the imported date fields to add a feature of your own to the application. For example, send a reminder email if the Contact has not visited the meetup site within 90 days.
